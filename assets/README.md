# Logo Placeholder

Por favor, coloque su logo de empresa en formato PNG en esta ubicación:
`assets/logo.png`

Dimensiones recomendadas: 200x200 píxeles o superior
Formato: PNG con fondo transparente preferiblemente

El logo se mostrará en:
- La ventana principal de la aplicación
- Los reportes PDF generados
- Los documentos Word exportados

Si no proporciona un logo, la aplicación funcionará normalmente sin mostrar la imagen.
