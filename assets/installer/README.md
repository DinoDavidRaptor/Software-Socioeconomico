# Assets para el Instalador - DINOS Tech

Esta carpeta contiene los recursos graficos para el instalador.

## Archivos Requeridos

### 1. icon.ico

- **Uso**: Icono del ejecutable y del instalador
- **Tamano**: Multiples resoluciones (16x16, 32x32, 48x48, 64x64, 128x128, 256x256)
- **Formato**: ICO (Windows Icon)
- **Como crear**: Usa herramientas como:
  - https://convertico.com/
  - https://icoconvert.com/
  - GIMP con plugin ICO

### 2. banner.bmp

- **Uso**: Banner lateral en el wizard del instalador
- **Tamano**: 164 x 314 pixeles
- **Formato**: BMP (24-bit)
- **Contenido sugerido**: Logo de DINOS Tech en vertical

### 3. header.bmp (wizard_header.bmp)

- **Uso**: Header en la parte superior del wizard
- **Tamano**: 150 x 57 pixeles (o proporcional)
- **Formato**: BMP (24-bit)
- **Contenido sugerido**: Logo pequeno + nombre de la app

### 4. splash.bmp (opcional)

- **Uso**: Pantalla de bienvenida
- **Tamano**: A gusto (tipicamente 600x400)
- **Formato**: BMP

## Estructura de Colores Sugerida

Basado en identidad DINOS Tech:

- Color primario: #2E86AB (Azul corporativo)
- Color secundario: #F0F4F8 (Gris claro)
- Acento: #E94560 (Rojo/naranja para alertas)

## Como Generar los Archivos

1. **Crea tu logo** en formato PNG con fondo transparente (256x256 minimo)
2. **Convierte a ICO** usando una herramienta online
3. **Crea los BMP** en cualquier editor de imagenes:
   - Photoshop
   - GIMP
   - Paint.NET
   - Canva (exportar como PNG, luego convertir)
